• Fakta:Okej och inte okej vid vab
  • En förälder som anmäler vård av sjukt barn får inte jobba.Försäkringskassan har bestämt att du får:
  • Prata om privata saker med kollegor och chefer som ringer.
  • Ringa jobbet och tala om hur länge du ska vara borta.
  • Skicka privata mejl och ringa privata samtal.
  • Om duändåblir pålagd jobb medan du vabbar menar Försäkringskassan att du ska stänga av telefonen och hänvisa till föräldraledighetslagen som säger att du inte har någon skyldighet att arbeta när barnetär sjukt.
  • Men det här får du inte:
  • Prata med din chef eller kollegor om direkt förvärvsarbete och diskussioner om arbetsuppgifter.
  • Skicka jobbmejl.
